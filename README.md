# fusion_de_perfiles
Ejercicios de JavaScript desarrollados durante el curso.

## Ejercicio: Spread y Desestructuración

Consolidación de dos objetos de perfil de usuario usando el operador
spread (`...`) y desestructuración, priorizando la fuente extendida.

### Conceptos practicados
- Operador spread (`...`)
- Desestructuración de objetos
- Combinación de objetos sin mutar los originales
